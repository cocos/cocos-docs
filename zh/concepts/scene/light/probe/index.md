# 光照探针

自 v3.7 开始，Cocos Creator 支持光照探针。

![preview](img/index.png)

光照探针是一种通过离线计算，将光照信息烘焙到贴图上，以较高的效率提升场景渲染品质的技术。

通常光照探针会由几部分构成：
- 插值算法
- 球谐基础
- 烘焙光照计算

对于此技术感兴趣的开发者，可移步 [光照探针理论基础](light-probe-theory.md)。
