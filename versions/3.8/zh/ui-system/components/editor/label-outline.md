# LabelOutline 组件参考

LabelOutline 组件会为所在节点的 Label 添加描边效果，不支持 BMFont 字体。

![](label/labeloutline.png)

点击 **属性检查器** 下方的 **添加组件** 按钮，然后从 **UI** 中选择 **LabelOutline**，即可添加 LabelOutline 组件到节点上。

**注意**: 3.8.2 版本开始 Label 组件已内置描边功能，无需再添加 LabelOutline 组件。

## LabelOutline 属性

| 属性 | 功能说明 |
| :-------------- | :----------- |
| Color | 字体发光边缘颜色 |
| Width | 字体发光边缘宽度 |
